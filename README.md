# AI Service Management System

The AI Service Management System is a modern IT ticketing platform designed to automate service desk operations using Artificial Intelligence. Built with Python, Streamlit, SQLite, and Ollama, the system allows users to submit support requests through a simple web interface while leveraging a local Large Language Model (Qwen3:8B) to automatically classify tickets by category, priority, and responsible support team.

The project aims to reduce manual triage efforts, improve response times, and provide intelligent routing of IT incidents. Future enhancements include incident tracking, SLA monitoring, AI-powered chat support, knowledge base search with RAG (Retrieval-Augmented Generation), analytics dashboards, and automated workflow management.

This project demonstrates how open-source AI technologies can be integrated into IT Service Management (ITSM) processes to create a scalable, cost-effective alternative to traditional enterprise service desk solutions.
